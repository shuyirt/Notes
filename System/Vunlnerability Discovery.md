### Bug

Type of Bugs

1. memory errors
2. race conditions
3. logic bugs / others

Memory Bugs

1. aka buffer overflow
2. coercing a program to write outside the bounds of an allocated object
3. "the write overflows" etc

Memory Erros has

1. a destination buffer being written to 

   1. stack allocated
   2. heap allocated
   3. global

2. with some source value 

   1. a static character
   2. input data
   3. non-input data

3. of some size

   1. fixed 
   2. variable 
   3. random
